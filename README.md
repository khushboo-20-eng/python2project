# 🗳️ Online Voting System

A simple **Online Voting System** built using **Python** 🐍 that allows users to register and vote for political parties.  
The program stores vote data in a text file and displays election results.


## 🛠️ Technologies Used

- Python 3
- File Handling
- Dictionary
- Loops & Conditions

---

## 🗳️ Available Parties

- BJP
- Congress
- AAP
